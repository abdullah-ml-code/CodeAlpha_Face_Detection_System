#  AI Face Detection System
A real-time image processing tool that identifies human faces in photos using **Computer Vision** techniques.

##  Key Features
- **Accurate Detection:** Powered by OpenCV's Haar Cascade algorithm.
- **Arabic Path Support:** Custom logic to handle file paths containing Arabic characters.
- **Modern UI:** Sleek and responsive interface built with `CustomTkinter`.

##  How it Works
The application converts images to grayscale and applies the **Haar Cascade Classifier** to locate facial features. Parameters like `minNeighbors` and `scaleFactor` have been optimized to balance speed and accuracy.

##  Requirements
- Python 3.x
- OpenCV (`opencv-python`)
- CustomTkinter
- Pillow
- NumPy
